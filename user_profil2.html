Product Registration

Registration steps and instructions:
  • 1) Supplier should have an approved user account.
  • 2) User should fill the product applications form and press “Next” button that leads to payment page after filling the forms for the required.
  • 3) User should fill the payment form and pay 265 EGP the data examination fees for the products,
  • 4) User should receive and request approval mail.
  • 5) Once the EOS finished the examination, if passed, user will receive mail with the required registration fees and payment page link.
  • 6) Once user pay the fees the product will appear in registered page.
